## Usage

- Download from [Releases](https://github.com/biyuehu/NeteaseLyricFetcher/releases)
- Install `Node.Js`
- Open the program and input NetEase Music ID or URL, e.g. `410446184`, `https://music.163.com/#/song?id=410446184`.

## Development

- Install `Just`
- Install `GCC`
- Install `Node.Js`
- Install `Zed`

Build:

```bash
just build
# Or:
# just b
```
